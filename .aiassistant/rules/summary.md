# AI Assistant Rules Summary - GuicedEE Website

- Use Markdown for all docs and follow host artifacts: `PACT.md`, `GLOSSARY.md`, `RULES.md`, `GUIDES.md`, `IMPLEMENTATION.md`.
- Behavioral agreements: precise technical English, maintain continuity, be transparent about limits, and iterate rather than reset.
- Technical commitments: reuse project naming conventions and link each artifact back to its parent layer for traceability.
- Document Modularity Policy: split large docs into topic files, remove monoliths, and update all indexes and references; keep kebab-case filenames under the same topic directory.
- Forward-Only Change Policy: no backward compatibility unless requested; update/remove all conflicting links and references in the same change; no legacy anchors or duplicate files.
