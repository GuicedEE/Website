## 🛠️ How to Use

When interacting with an AI agent in this repository, you can invoke these skills by mentioning their name (prefixed with `$` if the tool supports it) or by asking the agent to refer to the specific `SKILL.md` file.

### For Developers
If you are adding new functionality that requires specific patterns (e.g., a new Vert.x service), refer to the corresponding skill's `SKILL.md` for non-negotiable constraints and best practices.

### For AI Agents
AI agents should read the `SKILL.md` in the relevant directory to understand the domain-specific constraints, SPIs, and workflows. Metadata for UI integration is located in each skill's `agents/openai.yaml`.
