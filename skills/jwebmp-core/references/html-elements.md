# JWebMP Core HTML Elements Reference

Complete reference of all available HTML elements in JWebMP Core.

## Document Structure

- `Page` — Top-level HTML document (`<html>`)
- `Head` — Document head (`<head>`)
- `Body` — Document body (`<body>`)
- `Title` — Document title (`<title>`)
- `Meta` — Meta information (`<meta>`)
- `Link` — External resource link (`<link>`)
- `Script` — JavaScript (`<script>`)
- `Style` — CSS styles (`<style>`)

## Sectioning

- `Header` — Header section (`<header>`)
- `Footer` — Footer section (`<footer>`)
- `Nav` — Navigation section (`<nav>`)
- `Section` — Generic section (`<section>`)
- `Article` — Article content (`<article>`)
- `Aside` — Sidebar content (`<aside>`)
- `Main` — Main content (`<main>`)

## Headings

- `H1` through `H6` — Headings (`<h1>` to `<h6>`)

## Text Content

- `Div` — Division/container (`<div>`)
- `Span` — Inline container (`<span>`)
- `Paragraph` — Paragraph (`<p>`)
- `BlockQuote` — Block quotation (`<blockquote>`)
- `PreFormattedText` — Preformatted text (`<pre>`)
- `Code` — Code (`<code>`)
- `Strong` — Strong emphasis (`<strong>`)
- `Emphasis` — Emphasis (`<em>`)
- `Small` — Small text (`<small>`)
- `Mark` — Marked/highlighted text (`<mark>`)
- `Del` — Deleted text (`<del>`)
- `Ins` — Inserted text (`<ins>`)
- `Sub` — Subscript (`<sub>`)
- `Sup` — Superscript (`<sup>`)

## Lists

- `List` — Unordered/ordered list (`<ul>` or `<ol>`)
- `ListItem` — List item (`<li>`)
- `DescriptionList` — Description list (`<dl>`)
- `DescriptionTerm` — Description term (`<dt>`)
- `DescriptionDetails` — Description details (`<dd>`)

## Links & Media

- `Link` — Hyperlink (`<a>`)
- `Image` — Image (`<img>`)
- `Figure` — Figure with caption (`<figure>`)
- `FigureCaption` — Figure caption (`<figcaption>`)
- `Audio` — Audio (`<audio>`)
- `Video` — Video (`<video>`)
- `Source` — Media source (`<source>`)
- `Track` — Text track for media (`<track>`)
- `Canvas` — Graphics canvas (`<canvas>`)
- `SVG` — Scalable Vector Graphics (`<svg>`)

## Tables

- `Table` — Table (`<table>`)
- `TableCaption` — Table caption (`<caption>`)
- `TableHeaderGroup` — Table header group (`<thead>`)
- `TableBodyGroup` — Table body group (`<tbody>`)
- `TableFooterGroup` — Table footer group (`<tfoot>`)
- `TableRow` — Table row (`<tr>`)
- `TableHeaderCell` — Header cell (`<th>`)
- `TableDataCell` — Data cell (`<td>`)
- `TableColumnGroup` — Column group (`<colgroup>`)
- `TableColumn` — Column (`<col>`)

## Forms

- `Form` — Form (`<form>`)
- `FieldSet` — Field set (`<fieldset>`)
- `Legend` — Field set legend (`<legend>`)
- `Label` — Form label (`<label>`)
- `Button` — Button (`<button>`)
- `Select` — Select dropdown (`<select>`)
- `Option` — Select option (`<option>`)
- `OptionGroup` — Option group (`<optgroup>`)
- `TextArea` — Multi-line text input (`<textarea>`)
- `Output` — Output (`<output>`)
- `Progress` — Progress bar (`<progress>`)
- `Meter` — Meter/gauge (`<meter>`)

- `Input` — Generic input (`<input>`)
- `InputTextType` — Text input (`type="text"`)
- `InputPasswordType` — Password input (`type="password"`)
- `InputEmailType` — Email input (`type="email"`)
- `InputNumberType` — Number input (`type="number"`)
- `InputTelephoneType` — Telephone input (`type="tel"`)
- `InputURLType` — URL input (`type="url"`)
- `InputSearchType` — Search input (`type="search"`)
- `InputDateType` — Date input (`type="date"`)
- `InputDateTimeType` — DateTime input (`type="datetime-local"`)
- `InputTimeType` — Time input (`type="time"`)
- `InputWeekType` — Week input (`type="week"`)
- `InputMonthType` — Month input (`type="month"`)
- `InputColorType` — Color picker (`type="color"`)
- `InputRangeType` — Range slider (`type="range"`)
- `InputCheckBoxType` — Checkbox (`type="checkbox"`)
- `InputRadioType` — Radio button (`type="radio"`)
- `InputFileType` — File upload (`type="file"`)
- `InputSubmitType` — Submit button (`type="submit"`)
- `InputResetType` — Reset button (`type="reset"`)
- `InputButtonType` — Button (`type="button"`)
- `InputHiddenType` — Hidden input (`type="hidden"`)
- `InputImageType` — Image button (`type="image"`)
