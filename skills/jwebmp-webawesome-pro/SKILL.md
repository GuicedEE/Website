# JWebMP WebAwesome Pro

WebAwesome Pro integration for JWebMP with premium icons and features.

## Core Features

- **Premium Icons** — Extended icon collection including Thin, Light, and Duotone styles
- **Additional Styles** — Specialized style variations for professional branding
- **Advanced Features** — Premium utilities, advanced animations, and SVG optimization tools

## Quick Start

```java
import com.jwebmp.webawesome.icons.*;

Icon premiumIcon = new Icon()
    .setIconClass(WebAwesomeIcons.WA_CROWN)
    .setStyle(IconStyle.DUOTONE)
    .setSize(IconSize.XL);
```

## Installation

```xml
<dependency>
  <groupId>com.jwebmp.plugins</groupId>
  <artifactId>webawesome-pro</artifactId>
</dependency>
```
